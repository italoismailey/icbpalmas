# ⛪ Projetos para o Reino | Soluções Tecnológicas para Igrejas

Este repositório centraliza e demonstra projetos de software (sites, apps e ferramentas) desenvolvidos por mim para auxiliar no dia a dia da igreja onde congrego. O objetivo é unir tecnologia e serviço, criando ferramentas que facilitem a comunicação e a gestão ministerial.

---

## 🛠️ Portfólio de Projetos

Abaixo, os principais projetos desenvolvidos ou em fase de prototipagem:

### 🌐 Aplicações Web
* **Landing Page de Eventos:** Sites responsivos para conferências e congressos.
* **Portal Institucional:** Informações sobre cultos, departamentos e pedidos de oração.
* **Sistema de Escalas:** Dashboard para visualização de voluntários e funções semanais.

### 📱 Aplicativos e Mobile
* **App de Gestão de Células:** Acompanhamento de frequência e roteiros de estudo.
* **Guia de Membros:** Web-app para acesso rápido a avisos e manuais da igreja.

### ⚙️ Ferramentas e Automações
* **Automação de Relatórios:** Scripts em **Python** para processamento de dados administrativos.
* **Integração SQL:** Estruturação de bancos de dados para controle de patrimônio e membros.

---

## 🚀 Tecnologias Aplicadas

No desenvolvimento desses projetos, foco em tecnologias modernas e escaláveis:

* **Linguagens:** Python, JavaScript, SQL.
* **Banco de Dados:** PostgreSQL, Oracle (Tasy/MV) ou Firebase para soluções rápidas.
* **Cloud:** Implementações integradas com Azure e serviços de hospedagem web.
* **Data:** Análise de dados e métricas para apoio à liderança.

---

## 🎯 Objetivo do Repositório

1.  **Demonstração:** Servir como um portfólio de soluções técnicas reais.
2.  **Open Source:** Disponibilizar modelos que outras comunidades possam adaptar.
3.  **Aprimoramento:** Aplicar conceitos de **Análise e Desenvolvimento de Sistemas (ADS)** em um contexto prático e voluntário.
